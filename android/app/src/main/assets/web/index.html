קול הקראה

איך מוסיפים קולות למכשיר?

אנדרואיד: הגדרות → ניהול כללי / מערכת → רשימת שפות וקלט → פלט טקסט לדיבור → בחר מנוע Google Text-to-speech → התקנת נתוני קול → עברית → הורד (לעיתים יש כמה גרסאות קול).

אייפון/אייפד: הגדרות → נגישות → תוכן מדובר → קולות → עברית → הורד קולות נוספים (למשל גרסאות מורחבות).

לאחר ההורדה, חזור לכאן — הקולות החדשים יופיעו ברשימה.

פרשת השבוע

כשהמתג מכובה סדר פרשות השבוע הוא כפי הנהוג בארצנו הקדושה

תהלים יומי

כשהמתג פעיל, לחיצה על “תהלים יומי” תציג קודם את הפרקים האישיים שהגדרת, ואחריהם את החלוקה היומית.

תהלים לפי ימי הולדת

לכל אדם ייאמר הפרק כמספר שנת חייו (הגיל + 1), ויתחלף אוטומטית ביום ההולדת העברי. הפרקים יופיעו בכרטיסיית "תהלים אישי".

גיבוי ושחזור

הגיבוי שומר את כל ההגדרות (ימי הולדת, פרקים אישיים, סדר/הצגת כרטיסיות, גופנים, וסימוני המעקב) כקוד שאפשר לשמור במייל / Keep / Drive, ולשחזר במכשיר אחר.

אתר מכון גל עיני — galeinai.org.il

מָה מְאַחֵד אוֹתָנוּ? הַדֶּבֶק שֶּׁמְּחַבֵּר אוֹתָנוּ צָרִיךְ לִהְיוֹת הַתּוֹרָה. זֶהוּ הַחִבּוּר הָאֲמִתִּי. מֻמְלָץ מְאֹד - לִפְנֵי כְּשָׁנָה הִתְחַלְנוּ לִלְמֹד כָּל יוֹם פֶּרֶק תָּנָ"ךְ. אִם כֻּלָּם לוֹמְדִים אֶת פֶּרֶק הַתָּנָ"ךְ הַזֶּה, כַּמָּה שֶׁזֶּה כְּאִלּוּ חִיצוֹנִי - זֶה בְּהֶחְלֵט מְחַבֵּר. חוּץ מֵהַחִתָּ"ת שֶׁל הָרַבִּי הַקּוֹדֵם, הָרַבִּי הוֹסִיף גַּם רַמְבָּ"ם, וְזֶהוּ הַחִבּוּר בֵּין הַחֲסִידִים, הַלִּמּוּד הַיּוֹמִי הַזֶּה. בִּשְׁבִילֵנוּ אֲנַחְנוּ הוֹסַפְנוּ כַּמָּה דְּבָרִים לֹא קָשִׁים. לֹא מְסֻבָּךְ: פֶּרֶק תָּנָ"ךְ (אֲנַחְנוּ הַיּוֹם בְּסֵפֶר אִיּוֹב, סֵפֶר הַפְּסִיכוֹלוֹגְיָה הֲכִי חָשׁוּב בַּתָּנָ"ךְ), וּפֶרֶק מִשְׁנָה (שָׁבוּעַ הַבָּא מְסַיְּמִים חֲגִיגָה, סוֹף סֵדֶר מוֹעֵד, וּמַתְחִילִים סֵדֶר נָשִׁים). לְדַעְתִּי, מְאֹד מְאֹד רָצוּי שֶּׁכָּל מִי שֶּׁיּוֹשֵׁב כָּאן יִלְמַד גַּם אֶת הַחִתָּ"ת, וְגַם אֶת הַדְּבָרִים הָאֵלֶּה. זֶה בֶּאֱמֶת מְחַבֵּר, הַדְּבָרִים הָאֵלֶּה מְחַבְּרִים אוֹתָנוּ בְּיַחַד. [הָרַב יִצְחָק גִּינְזְבּוּרְג, מִתּוֹךְ שִׁעוּר כ"ד אִיָּר תשפ"ג]
